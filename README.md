## About The Project

This github repository explores open electric vehicle charging session data from different sources which can be used for non-commercial purposes. This research is presented in detail in: Amara-Ouali, Y.; Goude, Y.; Massart, P.; Poggi, J.-M.; Yan, H. A Review of Electric Vehicle Load Open Data and Models. Energies 2021, 14, 2233. [https://doi.org/10.3390/en14082233](https://doi.org/10.3390/en14082233) 


<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/yvenn-amara/EV-Load-Open-Data.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

## Usage

In 2. Scripts, you can find the R scripts used to preprocess and analyse the data.
In 3. Outputs, you can find the csv outputs generated after data preprocessing. 

Because of space limitations on GitHub we were not able to include the raw inputs as well as the Domestics UK preprocessed data but the links to these datasets are all available in the following paper: Amara-Ouali, Y.; Goude, Y.; Massart, P.; Poggi, J.-M.; Yan, H.
